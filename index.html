    Aplicação

    O Track Your Time vai além de uma simples agenda eletrônica, possuindo uma interface gráfica única o Track Your Time visa facilitar a vida do usuário de forma simples e elegante. Utilizando-se a sua função de agenda o Track Your Time armazena os dados fornecidos pelo usuário durante a semana a fim de disponibilizá-los de forma prática, demonstrando em quais atividades foi investido o seu tempo, tal qual, os seus horários disponíveis, tornando possível uma melhor distribuição e aproveitamento para o seu dia-a-dia.

    Plataformas:

    Android/NodeJS

    O aplicativo foi desenvolvido para o sistema Android com o servidor sendo implementado em NodeJS informações sobre essas tecnologias podem ser encontradas respectivamentes aqui(http://developer.android.com/sdk/index.html) e aqui(http://nodejs.org/).

    Ferramentas pra a qualidade de software

    FindBugs

    Umas das ferramentas que utilizamos para o controle de qualidade é o FindBugs. O findBugs é uma ferramenta que utiliza análise estática para encontrar erros em código Java. É um projeto criado pela Universidade de Maryland (The University of Maryland). Informações sobre essa ferramenta podem ser encontradas aqui (http://findbugs.sourceforge.net/)

    SonarQube

    SonarQube é uma plataforma de código aberto para inspeção contínua da qualidade do código. A ferramenta suporta algumas das seguintes features:

    -Suporta 25 + linguagens: Java, C / C + +, C #, PHP, Flex, Groovy, JavaScript, Python, PL / SQL, COBOL, etc. -Também pode ser usado no desenvolvimento do Android. -Oferece relatórios sobre código duplicado, padrões de codificação, testes unitários, cobertura de código, código complexo, possíveis bugs, comentários e design e arquitetura. -O histórico registra métricas e fornece gráficos de evolução ("máquina do tempo") e pontos de vista diferenciados. -Fornece análise totalmente automatizado:. Integra com Maven, Ant, Gradle e ferramentas de integração contínua (Atlassian Bamboo, Jenkins, Hudson, etc) -Integra-se com o ambiente de desenvolvimento Eclipse -Integra-se com ferramentas externas: JIRA, Mantis, LDAP, Fortify, etc. Informações sobre a ferramenta pode ser encontrada aqui(http://www.sonarqube.org/).

    Ferramentas de testes

    Robotium

    Robotium é um framework de teste de código aberto para escrever casos de testes automáticos para aplicativos Android. Com o apoio de Robotium, os desenvolvedores de casos de teste podem escrever funções ,cenários de teste do sistema e de aceitação. O Robotium pode ser usado tanto para aplicações de testes onde o código fonte está disponível ou aplicações onde somente o arquivo APK está disponível e os detalhes de implementação não são conhecidos.Informações sobre o framework podem se encontradas aqui(http://code.google.com/p/robotium/).
